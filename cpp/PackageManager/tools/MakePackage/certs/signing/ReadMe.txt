 === Brief Introduction to Package Manager package signing ===
 

These certificates are provided for PackageManager package signing TESTING ONLY!!
 

They were obtained by issuing a certificate request to this authority:
 http://www.frank4dd.com/sw/webcert/cgi-bin/buildrequest.cgi
 
 

The passphrase for all of them  is:  aaaaaaaa
 
 

The private key files can be used to sign a package using the 
 MakePackage.py script
 
 

The device running PackageManager must have installed the corresponding public key in its openssl certificate store.
